โปรเจคนี้เป็นเว็บแอปพลิเคชันสำหรับการประกาศขายอสังหาริมทรัพย์ พัฒนาด้วย React, Node.js, MySQL และใช้เครื่องมือ Vite สำหรับการพัฒนา

โปรเจคมีโครงสร้างโฟลเดอร์หลักๆ ดังนี้:

client: โฟลเดอร์สำหรับส่วนหน้าบ้าน (Frontend)
server: โฟลเดอร์สำหรับส่วนหลังบ้าน (Backend)
database: ไฟล์สำหรับฐานข้อมูล MySQL

โฟลเดอร์ client
โฟลเดอร์นี้ประกอบด้วยไฟล์และโฟลเดอร์ย่อยดังนี้:

.vite/: ไฟล์สำหรับการทำงานของ Vite
public/: ไฟล์ที่เป็นสาธารณะ
src/: โค้ดหลักสำหรับส่วนหน้าบ้าน
.eslintrc: ไฟล์ตั้งค่า ESLint
.gitignore: ไฟล์สำหรับกำหนดไฟล์ที่ไม่ต้องการให้ Git ติดตาม
index: ไฟล์หลักสำหรับการเริ่มต้นแอปพลิเคชัน
package.json: ไฟล์กำหนด package ที่ใช้
package-lock.json: ไฟล์ล็อคเวอร์ชันของ package
README.md: ไฟล์อธิบายโปรเจค
vite.config.js: ไฟล์ตั้งค่า Vite

โฟลเดอร์ server
โฟลเดอร์นี้ประกอบด้วยไฟล์และโฟลเดอร์ย่อยดังนี้:

uploads/: โฟลเดอร์สำหรับเก็บไฟล์ที่อัปโหลด
index.js: ไฟล์หลักสำหรับการเริ่มต้นเซิร์ฟเวอร์
package.json: ไฟล์กำหนด package ที่ใช้
package-lock.json: ไฟล์ล็อคเวอร์ชันของ package

การติดตั้งและการใช้งาน
- ติดตั้ง node_modules ในทั้งสองโฟลเดอร์ (client และ server) โดยการรันคำสั่ง npm install ภายในแต่ละโฟลเดอร์.

การเริ่มต้นใช้งาน
- เริ่มต้นเซิร์ฟเวอร์ด้วยคำสั่ง node index.js หรือ npm start ภายในโฟลเดอร์ server.
- เริ่มต้นส่วนหน้าบ้านด้วยคำสั่ง npm run dev ภายในโฟลเดอร์ client.
